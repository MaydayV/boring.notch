# OpenIsland

OpenIsland is a macOS notch companion app focused on a clean, compact interaction surface for music, system controls, and quick utilities.

## Requirements

- macOS 14 or later
- Xcode 16 or later

## Run Locally

1. Open `boringNotch.xcodeproj` in Xcode.
2. Select the `boringNotch` target.
3. Build and run with `Cmd + R`.

## Project Scope

This repository is maintained as a fresh OpenIsland codebase. Legacy project branding, release history, and repository references are intentionally removed.

## Contributing

- Keep changes focused and small.
- Verify the app builds before opening a PR.
